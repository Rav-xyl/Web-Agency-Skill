# /wsa:help — All Commands

```
/wsa:init          New project — profile, discovery, state files
/wsa:audit         Existing project — ingest, score, audit, report
/wsa:discuss N     Capture decisions before planning phase N
/wsa:plan N        Research + XML task specs for phase N
/wsa:execute N     Run all tasks in waves, commit per task
/wsa:verify N      User acceptance test, fix plans for failures
/wsa:next          Auto-detect state, confirm, run next step
/wsa:results       Monthly results report
/wsa:handoff       Client handoff checklist + credential transfer
/wsa:quick [task]  Ad-hoc task with XML spec + verify + commit
/wsa:status        Project state, roadmap, open items at a glance
/wsa:help          This
```

State files (root): PROJECT.md, STATE.md, REQUIREMENTS.md, ROADMAP.md
Planning files: .planning/phases/N-CONTEXT.md, N-PLAN.md, N-SUMMARY.md, N-UAT.md
Reports: .planning/reports/, .planning/baseline-[date].md
Reference files: ../references/
